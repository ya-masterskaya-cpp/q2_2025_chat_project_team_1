Detailed Description

Пространство имен, содержащее классы и функции для реализации графического интерфейса приложения.

Пространство имен gui инкапсулирует классы, отвечающие за отображение пользовательского интерфейса приложения, такие как окна, диалоги, элементы управления и обработчики событий. Оно обеспечивает визуальное взаимодействие пользователя с приложением и обработку действий, выполняемых пользователем.
