Detailed Description

Класс, представляющий главное окно приложения.

Реализует главное окно приложения с элементами управления для отправки сообщений, управления комнатами, отображения списка пользователей и настройками.

void gui::MainFrame::Load ( )

Загружает настройки окна из файла конфигурации.

Загружает ширину и высоту главного окна из файла конфигурации, используя элемент управления wxFileConfig. Если параметры не найдены, используются значения по умолчанию (300x300).
