Detailed Description

Заголовочный файл для обработки команд пользователя.

Определяет перечисление Command, функции ParseCommand и HandleCommand для разбора и выполнения команд, вводимых пользователем.

◆ HandleCommand()

bool HandleCommand ( Command cmd,
ChatClient & client )

Обрабатывает указанную команду, выполняя соответствующие действия.

В зависимости от полученной команды вызывает соответствующие методы класса ChatClient для выполнения операций, таких как регистрация, авторизация, отправка сообщений и т.д.

Parameters
cmdКоманда для выполнения.
clientСсылка на объект ChatClient, используемый для выполнения команд.
Returns
true, если обработка команды прошла успешно, false, если нужно завершить работу приложения.

◆ ParseCommand()

Command ParseCommand ( const std::string & input)

Разбирает введенную пользователем строку и определяет команду.

Сопоставляет введенную строку с предопределенным списком команд и возвращает соответствующее значение перечисления Command.

Parameters
inputСтрока, введенная пользователем.
Returns
