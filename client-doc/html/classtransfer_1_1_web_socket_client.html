Detailed Description

Класс, реализующий WebSocket-клиента.

Устанавливает и управляет WebSocket-соединением для обмена сообщениями с сервером. Поддерживает установку обработчиков для различных событий WebSocket, таких как получение сообщения, открытие соединения, закрытие соединения и возникновение ошибки.
