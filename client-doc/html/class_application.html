Detailed Description

Наследник базового класса для всех приложений wxWidgets.

wxApp является основой любого приложения wxWidgets. Он управляет главным циклом событий, обрабатывает пользовательский ввод и предоставляет точку входа для инициализации приложения в функции OnInit().

Инициализация главного окна приложения

вызывается при запуске приложения, сразу после создания экземпляра вашего класса wxApp. Здесь вы создаете главное окно, инициализируете другие компоненты и выполняете любую другую настройку, необходимую для запуска вашего приложения.

Returns
true, если инициализация прошла успешно, и false, если произошла ошибка (в этом случае приложение завершится).
