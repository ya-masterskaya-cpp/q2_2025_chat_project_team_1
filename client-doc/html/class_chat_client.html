Detailed Description

Клиент для чат-сервера, реализующий взаимодействие через REST API и WebSocket.

Класс ChatClient предоставляет методы для аутентификации, управления пользователями, отправки сообщений и работы с комнатами чата. Он использует REST API для основных операций и WebSocket для обмена сообщениями в реальном времени.
