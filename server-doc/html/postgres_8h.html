Detailed Description

Заголовочный файл, определяющий интерфейс для работы с базой данных PostgreSQL.

Содержит объявления пространств имен, структур данных и классов, предназначенных для организации взаимодействия с базой данных PostgreSQL, включая пулы соединений, репозитории для пользователей, комнат, сообщений и участников комнат.
