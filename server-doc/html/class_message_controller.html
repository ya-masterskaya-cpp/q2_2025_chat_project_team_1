Detailed Description

Контроллер для обработки запросов, связанных с сообщениями.

Реализует REST API для отправки, сохранения в БД и получения сообщений. Важно! При запросе на отправку SendMessage сохранение не выполняется, сохранение происходит по запросу UploadMessage.
