Detailed Description

template<typename Tag>
class util::TaggedUUID< Tag >

Шаблон класса для представления UUID с тегом.

Наследуется от класса Tagged, предоставляя возможность связать UUID с определенным тегом (типом), что может быть полезно для различения UUID в разных контекстах.
