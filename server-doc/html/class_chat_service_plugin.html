Detailed Description

Плагин для доступа к сервису управления чатом (ChatService).

Этот плагин обеспечивает глобальный доступ к сервису ChatService, который управляет аутентификацией, пользователями, комнатами и сообщениями в чате. Плагин создает экземпляр ChatService при инициализации и предоставляет статический метод для доступа к этому экземпляру.
