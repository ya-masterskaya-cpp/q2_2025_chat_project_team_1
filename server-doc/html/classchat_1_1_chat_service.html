Detailed Description

Предоставляет интерфейс для управления чатом, включая аутентификацию, управление пользователями и комнатами.

Класс ChatService предоставляет методы для аутентификации пользователей, управления комнатами чата и обработки сообщений. Он использует IRCDBWrapper для доступа к базе данных и TokenManager для управления токенами.
