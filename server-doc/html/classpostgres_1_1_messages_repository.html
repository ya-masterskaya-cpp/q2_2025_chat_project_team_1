Detailed Description

Класс для управления записями сообщений в базе данных.

Предоставляет методы для сохранения, загрузки последних сообщений, загрузки страницы сообщений и удаления записей сообщений в базе данных. Использует транзакцию pqxx::work для взаимодействия с базой данных.
