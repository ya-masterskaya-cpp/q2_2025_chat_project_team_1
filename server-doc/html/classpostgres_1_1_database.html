Detailed Description

Класс для управления подключением к базе данных.

Этот класс предоставляет интерфейс для получения соединений с базой данных из пула соединений. Он инкапсулирует пул соединений и предоставляет метод для получения транзакции, обернутой в ConnectionPool::ConnectionWrapper.
