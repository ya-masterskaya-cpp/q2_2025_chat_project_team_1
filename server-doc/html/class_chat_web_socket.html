Detailed Description

Класс, реализующий WebSocket-соединение для чата.

Этот класс обрабатывает входящие сообщения от клиентов, устанавливает новые соединения, обрабатывает закрытие соединений и рассылает сообщения между подключенными клиентами. Он использует мьютекс для обеспечения потокобезопасности.
