Detailed Description

template<typename Value, typename Tag>
class util::Tagged< Value, Tag >

Шаблон класса для добавления тега к значению.

Позволяет связать значение определенного типа с тегом (типом), что может быть полезно для различения значений одного и того же типа в разных контекстах.
